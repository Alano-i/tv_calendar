# 💌 追剧日历 - 修改版
MR插件，根据 or3ki 大佬的版本修改版，加了日志的细节输出和前端页面的调整。

根据订阅的剧集，查看未来7天的更新计划。
- 对于配置了MR访问地址的用户，点击微信推送内容可跳转到日历页面。
- 点击左下角问号，进入日历

## 使用方法
- 将 `tv_calendar` 文件夹放到 `Plugins` 文件夹，重启MR。
